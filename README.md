# NCAA Football Sportsbook Accuracy

A data analysis and visualization comparing NCAA football sportsbook odds to results during the 2023-24 season

---

## Overview

This project analyzes the difference between sportsbook lines and over/under predictions and the actual outcome of games. The goal is to identify prediction accuracy among notable sportsbooks, including William Hill, Bovada, DraftKings, and ESPN Bet. 

---

## Features

- Exploratory data analysis with visualizations
- Normal distribution
- Summary statistics
- Introductory and conclusion discussion
  
---

## Dataset

Source: https://collegefootballdata.com/

Contains ~3000 2024 NCAA football betting lines 

Important variables:
- `LineProvider`
- `Spread`
- `OverUnder`
- `HomeScore`
- `AwayScore`
